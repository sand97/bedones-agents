## Le paradoxe de la promo réussie

Vous lancez une promo. Elle marche au-delà de vos espérances. 300 messages WhatsApp en 2 heures. Votre équipe est noyée. Les clients attendent 3h pour une réponse. Certains annulent. Votre stock part en double parce que personne ne suit. Bref : **la promo qui devait vous faire +50 % vous coûte 20 % en réputation**. Voici comment l'éviter.

## J-7 : préparer le stock et le catalogue

- Vérifier les **quantités réelles** en magasin
- Geler les produits qui partent trop vite (n'inclure dans la promo que ce que vous pouvez livrer)
- Mettre à jour les **photos et prix** dans tous vos canaux (TikTok, Instagram, WhatsApp)
- Briefer l'équipe livreurs sur la période

## J-3 : préparer les réponses types

Listez les 20 questions qui vont arriver :

- *"C'est valable jusqu'à quand ?"*
- *"Vous livrez à Bouaké ?"*
- *"Je peux payer en Wave ?"*
- *"Vous avez encore en taille 42 ?"*

Préparez des **réponses claires, courtes, prêtes à coller**. Mieux : injectez-les dans votre agent IA pour qu'il les serve automatiquement.

## J-1 : tester le tunnel complet

Faites une commande test :

1. Cliquez sur votre annonce
2. Ouvrez WhatsApp
3. Demandez le produit
4. Payez en Mobile Money
5. Vérifiez la confirmation

Si une étape coince à vide, elle explosera sous charge.

## Jour J : le triage par canal

Pendant le pic, votre erreur n°1 sera de vouloir tout traiter à la main. Répartissez :

- **Agent IA** → questions de prix, dispo, livraison, paiement
- **Humain 1** → négociations, gros paniers, VIP
- **Humain 2** → SAV, réclamations, problèmes de paiement

Un outil comme **Bedones Moderator** route automatiquement chaque conversation vers le bon traitement, selon des règles que vous définissez en amont.

## Sécuriser les paiements

Les pics génèrent aussi des **fraudes** : faux justificatifs Orange Money, captures retouchées Wave. Règles d'or :

- Ne JAMAIS libérer la commande sans confirmation API du paiement
- Croiser numéro de téléphone + montant + référence
- En cas de doute, demander une **deuxième vérification** avant expédition

## Le débrief J+1

Le lendemain, prenez 30 minutes pour noter :

- Combien de messages reçus / traités
- Combien de paniers récupérés / perdus
- Quelle question revenait le plus
- Où l'équipe a été dépassée

Ces 4 chiffres pilotent votre prochaine promo, et la suivante, et la suivante.

## La règle d'or des pics

Plus le pic est gros, plus vous avez besoin que **80 % des conversations se traitent sans vous**. C'est exactement le rôle d'un agent IA branché sur votre catalogue : il absorbe le volume, vous gardez la marge et le sommeil.
